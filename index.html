            <h2 class="font-serif text-3xl font-bold text-gray-800 mb-6">Introducción</h2>

            <p class="text-lg text-gray-700 leading-relaxed mb-6">

              Northrop Frye (<a href="https://macblog.mcmaster.ca/fryeblog/" class="text-primary hover:underline" target="_blank">1912-1991</a>)

              revolucionó la comprensión de la Biblia al proponer una lectura que trasciende los enfoques puramente religiosos o históricos.

              En su obra maestra <em>El gran código: La Biblia y la literatura</em>, Frye establece que la Biblia es el

              <strong>"Gran Código de la cultura occidental"</strong>, una fuente primaria de mitología y arquetipos que ha moldeado

              la literatura y el arte durante siglos.

            </p>

            <p class="text-lg text-gray-700 leading-relaxed">

              Su interpretación se basa en el análisis de su <strong>estructura quíasica</strong> (un "doble espejo" entre el Antiguo

              y Nuevo Testamento), su lenguaje único de <strong>kerygma</strong> (una proclamación que trasciende lo literal y revela

              una realidad espiritual), y su método de <strong>tipología</strong>, que revela identidades subyacentes más allá de

              los eventos históricos.

            </p>

          </div>

          <div class="quote-highlight p-6 rounded-lg">

            <blockquote class="font-serif text-xl italic text-gray-800 mb-4">

              "La Biblia es el Gran Código del Arte"

            </blockquote>

            <cite class="text-sm text-gray-600">— William Blake, citado por Frye</cite>

            <p class="text-sm text-gray-700 mt-4">

              Esta cita de Blake, adoptada por Frye como título de su obra, encapsula la visión de la Biblia como

              el código primigenio que ha informado la imaginación occidental.

            </p>

          </div>

        <h2 class="font-serif text-4xl font-bold text-gray-800 mb-12">1. El Gran Código: La Biblia como Unidad y Fundamento Cultural</h2>


        <div class="grid grid-cols-1 lg:grid-cols-2 gap-12 mb-16">

          <div>

            <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">1.1 La Biblia como "Gran Código" de la Cultura Occidental</h3>

            <p class="text-gray-700 leading-relaxed mb-4">

              Northrop Frye establece que la Biblia no es simplemente un texto religioso, sino el

              <strong>"Gran Código que ha moldeado la imaginación y la cultura occidental"</strong>

              <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[502]</a>.

              El título mismo del libro proviene de una cita de William Blake, quien afirmó que la Biblia es "el Gran Código del Arte".

            </p>

            <p class="text-gray-700 leading-relaxed mb-4">

              En lugar de abordar la Biblia desde una perspectiva teológica o histórica, Frye la examina como un crítico literario,

              interesado en su unidad interna y su poder como un sistema de mitos y metáforas

              <a href="https://www.dialoguejournal.com/articles/the-great-code-revealed-bible-and-literature-by-northrop-frye/" class="text-primary hover:underline" target="_blank">[519]</a>.

            </p>

          </div>

          <div>

          </div>

        </div>

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">1.2 Unidad Estructural y Narrativa</h3>


          <div class="mb-8">

            <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">1.2.1 Estructura Quíasica o de "Doble Espejo"</h4>

            <p class="text-gray-700 leading-relaxed mb-4">

              El libro de Frye se divide en dos partes: <strong>"El orden de las palabras" y "El orden de los tipos"</strong>

              <a href="https://www.dialoguejournal.com/articles/the-great-code-revealed-bible-and-literature-by-northrop-frye/" class="text-primary hover:underline" target="_blank">[519]</a>.

              La primera parte explora los principios básicos de interpretación a través de los conceptos de lenguaje, mito, metáfora y tipología.

              La segunda parte revisita estos mismos cuatro temas, pero en orden inverso.

            </p>

            <div class="quote-highlight p-6 rounded-lg mb-6">

              <p class="font-serif text-lg italic text-gray-800 mb-2">

                "Esta disposición crea una estructura quíasica, o de 'doble espejo', donde cada mitad del libro refleja y revela el significado de la otra."

              </p>

              <cite class="text-sm text-gray-600">

                — <a href="https://macblog.mcmaster.ca/fryeblog/what-the-great-code-is-and-does/" class="text-primary hover:underline" target="_blank">[524]</a>

              </cite>

            </div>

          </div>


          <div class="mb-8">

            <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">1.2.2 Coherencia entre el Antiguo y el Nuevo Testamento</h4>

            <p class="text-gray-700 leading-relaxed mb-4">

              La unidad de la Biblia se manifiesta en la coherencia entre los dos testamentos. A través de su método tipológico,

              Frye demuestra que los dos testamentos no son colecciones independientes de textos, sino partes de una sola narrativa

              que se refleja y se completa a sí misma

              <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[502]</a>.

            </p>

            <div class="quote-highlight p-6 rounded-lg">

              <p class="font-serif text-lg italic text-gray-800 mb-2">

                "El Antiguo Testamento se revela en el Nuevo, y el Nuevo Testamento está oculto en el Antiguo,

                creando una 'doble mirada' en la que cada parte refleja la otra."

              </p>

              <cite class="text-sm text-gray-600">

                — <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[37]</a>

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">1.3 La Biblia como Fuente de Mitología Occidental</h3>


          <div class="grid grid-cols-1 md:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">1.3.1 Influencia en la Literatura y el Arte Seculares</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye sostiene que la Biblia ha sido la fuente primaria de la mitología occidental, proporcionando los arquetipos

                y los símbolos que han alimentado la imaginación de los artistas y escritores a lo largo de los siglos.

                En <em>The Secular Scripture</em>, Frye explora cómo la literatura secular ha recreado y secularizado los mitos bíblicos

                <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[502]</a>.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">1.3.2 La Biblia como "Mito del Hombre Occidental"</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Para Frye, la Biblia es el <strong>"mito del hombre occidental"</strong>, una narrativa que define la identidad

                y el destino de la humanidad desde una perspectiva occidental. A diferencia de otros mitos que se centran en dioses

                y héroes, el mito bíblico es una historia sobre la humanidad

                <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[516]</a>.

              </p>

        <h2 class="font-serif text-4xl font-bold text-gray-800 mb-12">2. Kerygma: El Lenguaje Único de la Biblia</h2>


        <div id="kerygma-definicion" class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">2.1 Definición de Kerygma como "Proclamación"</h3>


          <div class="grid grid-cols-1 lg:grid-cols-3 gap-8">

            <div class="lg:col-span-2">

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye adopta el término griego <em>kerygma</em>, que significa "proclamación" o "predicación", y le otorga un significado

                mucho más profundo y específico en el contexto del lenguaje bíblico

                <a href="https://macblog.mcmaster.ca/fryeblog/what-the-great-code-is-and-does/" class="text-primary hover:underline" target="_blank">[384]</a>.

                Para él, <em>kerygma</em> no es simplemente el contenido del mensaje cristiano, sino la forma lingüística única

                a través de la cual se transmite la revelación divina.

              </p>

              <p class="text-gray-700 leading-relaxed mb-6">

                Este lenguaje no es argumentativo ni persuasivo en el sentido retórico convencional; más bien, es un acto de habla

                que crea una nueva realidad para el oyente. Dios y Jesús, en sus expresiones de preocupación, utilizan metáforas,

                pero no se comprometen en argumentaciones.

              </p>

            </div>

            <div class="quote-highlight p-6 rounded-lg">

              <p class="font-serif text-lg italic text-gray-800 mb-2">

                "El kerygma es un acto místico y mistagógico que tiene el poder de iniciar al oyente en una realidad espiritual superior."

              </p>

              <cite class="text-sm text-gray-600">

                — <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[437]</a>

              </cite>

            </div>

          </div>


          <div class="grid grid-cols-1 md:grid-cols-2 gap-8 mt-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">2.1.1 Kerygma como Acto Místico y Mistagógico</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Para Frye, el <em>kerygma</em> es mucho más que una simple proclamación verbal; es un <strong>acto místico y mistagógico</strong>

                que tiene el poder de iniciar al oyente en una realidad espiritual superior. El término "mistagógico" se refiere a la

                iniciación en los misterios.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">2.1.2 Revelación de una Realidad Espiritual</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                El <em>kerygma</em> es el principal mecanismo a través del cual la Biblia revela una realidad espiritual que trasciende

                el mundo físico. El lenguaje <em>kerygmatico</em> no describe esta realidad objetivamente; más bien, la "revela" al traerla

                a la existencia en la imaginación y la conciencia del oyente

                <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[75]</a>.

              </p>

            </div>

          </div>

        </div>


        <div class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">2.2 Kerygma vs. Lenguaje Literal y Descriptivo</h3>


          <div class="mb-8">

            <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">2.2.1 Verdad Poética vs. Verdad Literal</h4>

            <p class="text-gray-700 leading-relaxed mb-4">

              Frye argumenta que la Biblia no debe ser leída como un documento histórico o científico que pretende ofrecer una

              descripción literal y objetiva de los hechos. Intentar leer la Biblia de esta manera es caer en el "literalismo",

              una forma de anti-intelectualismo que malinterpreta la naturaleza del lenguaje bíblico.

            </p>

          </div>


          <div class="mb-8">

            <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">2.2.2 Transcendencia de la División Sujeto/Objeto</h4>

            <p class="text-gray-700 leading-relaxed mb-4">

              Una de las características más importantes del <em>kerygma</em> es su capacidad para <strong>trascender la división entre sujeto y objeto</strong>

              que caracteriza al lenguaje descriptivo. El <em>kerygma</em> es un lenguaje que "nos posee", más que nosotros poseerlo,

              y en el que la separación entre el habla activa y la recepción de la palabra desaparece

              <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[516]</a>.

            </p>

          </div>

        </div>


        <div id="kerygma-fases">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">2.3 Kerygma en el Marco de las Fases del Lenguaje de Vico</h3>


          <div class="grid grid-cols-1 lg:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">2.3.1 Posicionamiento de Kerygma como Fase Superior</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Para situar el <em>kerygma</em> en el contexto de la historia del lenguaje, Frye se sirve de la teoría de las tres fases

                del lenguaje de Giambattista Vico: la fase poética, la fase heroica y la fase vulgar

                <a href="https://macblog.mcmaster.ca/fryeblog/2009/09/24/the-phases-and-modes-of-language/" class="text-primary hover:underline" target="_blank">[525]</a>.

              </p>

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye propone el <em>kerygma</em> como una cuarta forma de expresión, o incluso como una quinta fase que se sitúa

                <strong>"al otro lado de lo poético"</strong>

                <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[535]</a>.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">2.3.2 Contraste con la Desmitologización de Bultmann</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                La postura de Frye se ve reforzada por su rechazo a la teoría de la <strong>"demitologización"</strong> del teólogo Rudolf Bultmann.

                Mientras Bultmann argumentaba que el mensaje del Nuevo Testamento debía ser separado de su envoltura mítica,

                Frye insiste en que el mito es el lenguaje apropiado y necesario para expresar las verdades del <em>kerygma</em>

                <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[530]</a>.

              </p>

            </div>

        <h2 class="font-serif text-4xl font-bold text-gray-800 mb-12">3. Orden de Tipos: La Tipología como Método Interpretativo</h2>


        <div id="estructura-unificadora" class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">3.1 Tipología como Estructura Unificadora de la Biblia</h3>


          <div class="grid grid-cols-1 lg:grid-cols-2 gap-8 mb-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">3.1.1 Relación entre Tipo y Antitipo</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                La tipología es el método interpretativo que Frye utiliza para demostrar la unidad estructural de la Biblia.

                Este método establece una relación de correspondencia entre los eventos y personajes del Antiguo Testamento

                (los "tipos") y su cumplimiento en el Nuevo Testamento (los "antitypos")

                <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[502]</a>.

              </p>

              <p class="text-gray-700 leading-relaxed">

                La relación entre el tipo y el antitipo no es simplemente una cuestión de predicción y cumplimiento,

                sino de una <strong>"doble mirada" en la que cada testamento refleja y da sentido al otro</strong>

                <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[37]</a>.

              </p>

            </div>

            <div>

              <img src="https://kimi-img.moonshot.cn/pub/icon/spinner.svg" alt="Typological interpretation of Old and New Testament biblical events" class="w-full h-64 object-cover rounded-lg shadow-lg" size="medium" aspect="wide" query="biblical typology old new testament" referrerpolicy="no-referrer" />

            </div>

          </div>


          <div class="quote-highlight p-6 rounded-lg mb-8">

            <p class="font-serif text-lg italic text-gray-800 mb-2">

              "El Antiguo Testamento se revela en el Nuevo, y el Nuevo Testamento está oculto en el Antiguo,

              creando una estructura de eco y reverberación que une los dos testamentos en una sola narrativa."

            </p>

            <cite class="text-sm text-gray-600">

              — <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[502]</a>

            </cite>

          </div>

        </div>


        <div id="identidades-subyacentes" class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">3.2 Tipología y la Tensión entre Literatura y Teología</h3>


          <div class="grid grid-cols-1 md:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">3.2.1 Uso de la Tipología como Herramienta Literaria</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Aunque Frye insiste en que su enfoque de la Biblia es literario y no teológico, su uso de la tipología lo sitúa

                en una zona de tensión entre ambos campos. Frye adopta la tipología como una herramienta para analizar la estructura

                y el significado de la Biblia como un texto literario.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">3.2.2 Reconocimiento de las Raíces Teológicas</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                A pesar de su enfoque literario, Frye reconoce que la tipología no puede disociarse completamente de su fundamento teológico.

                Los críticos señalan que su hermenéutica bíblica comparte principios fundamentales con muchos teólogos cristianos modernos

                <a href="https://dialnet.unirioja.es/descarga/articulo/1271431.pdf" class="text-primary hover:underline" target="_blank">[1]</a>.

              </p>

            </div>

          </div>

        </div>

      </section>

      <!-- Section 4: Mitopoesia -->

      <section id="mitopoesia" class="max-w-7xl mx-auto px-6 py-16">

        <h2 class="font-serif text-4xl font-bold text-gray-800 mb-12">4. Mitopoesía vs. Historia: La Biblia como Mito, no como Crónica</h2>


        <div id="narrativa-mitica" class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">4.1 La Biblia como Narrativa Mítica</h3>


          <div class="grid grid-cols-1 lg:grid-cols-2 gap-8 mb-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">4.1.1 Énfasis en el Significado sobre la Facticidad</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Para Frye, la Biblia debe ser entendida primordialmente como una narrativa mítica, no como un documento histórico.

                El término "mito", en su uso, no implica falsedad, sino una secuencia de palabras cargada de una seriedad y una importancia especiales

                <a href="https://www.dialoguejournal.com/articles/the-great-code-revealed-bible-and-literature-by-northrop-frye/" class="text-primary hover:underline" target="_blank">[69]</a>.

              </p>

              <p class="text-gray-700 leading-relaxed">

                El valor de la Biblia no radica en su veracidad histórica, sino en su capacidad para transmitir verdades profundas

                sobre la condición humana y su relación con lo divino.

              </p>

            </div>

            <div class="quote-highlight p-6 rounded-lg">

              <p class="font-serif text-lg italic text-gray-800 mb-2">

                "Comparar los mitos de diluvio con inundaciones históricas es menos fructífero que compararlos con otros mitos de diluvio,

                ya que su significado reside en su estructura simbólica, no en su facticidad."

              </p>

              <cite class="text-sm text-gray-600">

                — <a href="https://tntlab.carterschool.gmu.edu/learn/narrative-transformation-resources/narrative-themography/myth/" class="text-primary hover:underline" target="_blank">[78]</a>

              </cite>

            </div>

          </div>


          <div class="mb-8">

            <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">4.1.2 "Heilsgeschichte" vs. "Weltgeschichte"</h4>

            <p class="text-gray-700 leading-relaxed mb-4">

              Frye distingue entre dos tipos de historia: la <em>Heilsgeschichte</em> (historia de la salvación) y la

              <em>Weltgeschichte</em> (historia del mundo). La Biblia pertenece a la primera categoría.

            </p>

            <p class="text-gray-700 leading-relaxed">

              La <em>Weltgeschichte</em> es la historia de los eventos tal como ocurrieron, una crónica de hechos sin un propósito trascendental.

              La <em>Heilsgeschichte</em>, por otro lado, es una narrativa de redención, una historia que tiene un sentido y una dirección

              <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[37]</a>.

            </p>

          </div>

        </div>


        <div id="critica-historico">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">4.2 Crítica al Enfoque Histórico-Crítico</h3>


          <div class="grid grid-cols-1 md:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">4.2.1 Rechazo de la Lectura como Documento Histórico</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye rechaza el enfoque histórico-crítico de la Biblia, que busca establecer la veracidad histórica de sus relatos

                a través del análisis de fuentes y la comparación con otros documentos históricos. Para Frye, este enfoque es inadecuado

                porque ignora la naturaleza mítica y simbólica de la Biblia.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">4.2.2 Defensa de un Enfoque Literario y Simbólico</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                En lugar del enfoque histórico-crítico, Frye defiende un enfoque literario y simbólico de la Biblia.

                Este enfoque se centra en el análisis de la estructura narrativa, los patrones de imágenes y los símbolos de la Biblia,

                con el objetivo de revelar su unidad y coherencia como una obra de arte.

              </p>

            </div>

          </div>

        </div>

      </section>


      <div class="section-divider max-w-7xl mx-auto"></div>


      <!-- Section 5: Arquetipos -->

      <section id="arquetipos" class="max-w-7xl mx-auto px-6 py-16">

        <h2 class="font-serif text-4xl font-bold text-gray-800 mb-12">5. Imagen y Arquetipo: Los Símbolos Visuales de la Biblia</h2>


        <div id="sistemas-imagineria" class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">5.1 Sistemas de Imaginería Unificados</h3>


          <div class="grid grid-cols-1 lg:grid-cols-2 gap-8 mb-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">5.1.1 Imaginería Apocalíptica vs. Imaginería Demoníaca</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye identifica dos sistemas de imaginería opuestos y unificados en la Biblia: la <strong>imaginería apocalíptica</strong>

                y la <strong>imaginería demoníaca</strong>. La imaginería apocalíptica representa el mundo de la creación, el paraíso y la redención,

                mientras que la imaginería demoníaca representa el mundo de la caída, el caos y la destrucción.

              </p>

            </div>

            <div>

              <img src="https://kimi-img.moonshot.cn/pub/icon/spinner.svg" alt="Biblical imagery contrast: heavenly paradise versus chaotic destruction" class="w-full h-64 object-cover rounded-lg shadow-lg" size="medium" aspect="wide" query="biblical paradise and chaos contrast" referrerpolicy="no-referrer" />

            </div>

          </div>


          <div class="quote-highlight p-6 rounded-lg mb-8">

            <p class="font-serif text-lg italic text-gray-800 mb-2">

              "Estos dos sistemas de imaginería no son simplemente opuestos, sino que están intrínsecamente relacionados.

              La imaginería demoníaca es una perversión de la imaginería apocalíptica, y la imaginería apocalíptica

              es la restauración de la imaginería demoníaca."

            </p>

          </div>

        </div>


        <div id="naturaleza-simbolo">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">5.2 La Naturaleza como Símbolo</h3>


          <div class="grid grid-cols-1 md:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">5.2.1 La Creación como Orden Divino</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                En la Biblia, la naturaleza no es simplemente un fondo para la acción humana, sino un símbolo del orden divino.

                La creación, tal como se describe en el Génesis, es un acto de dar forma y orden al caos.

              </p>

              <p class="text-gray-700 leading-relaxed">

                El jardín del Edén es el símbolo supremo de esta armonía entre la naturaleza y lo divino

                <a href="https://classicalchristian.org/classis/the-bible-northrop-frye-classical-christian-education-part-i/" class="text-primary hover:underline" target="_blank">[187]</a>.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">5.2.2 La Caída como Caos y Desorden</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                La caída del hombre no es simplemente un evento histórico, sino un símbolo de la entrada del caos y el desorden en el mundo.

                La naturaleza, que antes era un reflejo del orden divino, se convierte en un lugar de alienación y sufrimiento.

              </p>

              <p class="text-gray-700 leading-relaxed">

                El desierto, con su sequedad y su soledad, se convierte en el símbolo supremo de la naturaleza caída.

              </p>

            </div>

          </div>

        </div>

      <section id="lenguaje-prosaico" class="max-w-7xl mx-auto px-6 py-16">

        <h2 class="font-serif text-4xl font-bold text-gray-800 mb-12">6. La Palabra y la Prosa: La Estructura Lingüística de la Biblia</h2>


        <div class="grid grid-cols-1 lg:grid-cols-2 gap-12 mb-16">

          <div>

            <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">6.1 El Lenguaje de la Biblia como "Prosa"</h3>


            <div class="mb-8">

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">6.1.1 Actos de Habla Prosaicos</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye argumenta que el lenguaje de la Biblia es esencialmente prosaico, en el sentido de que está compuesto por actos de habla

                que tienen un propósito específico. Los actos de habla prosaicos en la Biblia incluyen mandamientos, exhortaciones, oráculos y proclamaciones.

              </p>

              <p class="text-gray-700 leading-relaxed">

                La Biblia es una obra de "literatura plus", un texto que posee una gravedad y una seriedad que lo distinguen de la ficción puramente hipotética

                <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[179]</a>.

              </p>

            </div>

          </div>

          <div>

            <img src="https://kimi-img.moonshot.cn/pub/icon/spinner.svg" alt="Ancient biblical manuscript with calligraphy" class="w-full h-80 object-cover rounded-lg shadow-lg" size="medium" aspect="wide" style="photo" query="ancient biblical manuscript calligraphy" referrerpolicy="no-referrer" />

          </div>

        </div>


        <div class="mb-8">

          <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">6.1.2 Distinción entre Sentido Literal y Sentido Verbal</h4>

          <p class="text-gray-700 leading-relaxed mb-4">

            Frye hace una distinción fundamental entre el sentido literal y el sentido verbal de la Biblia.

            El sentido literal se refiere a la correspondencia entre las palabras y los hechos históricos.

            El sentido verbal se refiere al significado que surge de las conexiones y patrones internos del texto.

          </p>

          <p class="text-gray-700 leading-relaxed">

            Frye argumenta que el verdadero sentido "literal" de la Biblia es su sentido verbal, no su referencia a eventos históricos externos.

          </p>

        </div>


        <div class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">6.2 La Narrativa de la Biblia: De la Creación al Apocalipsis</h3>


          <div class="grid grid-cols-1 md:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">6.2.1 Estructura Narrativa Circular</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                La narrativa de la Biblia, según Frye, tiene una estructura circular que va desde la creación hasta el apocalipsis.

                El libro comienza con la creación del mundo en el Génesis y termina con la creación de un nuevo cielo y una nueva tierra en el Apocalipsis.

              </p>

              <p class="text-gray-700 leading-relaxed">

                Esta estructura circular sugiere que la historia no es una línea recta, sino un ciclo de caída y redención.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">6.2.2 El "Verbo" como Motor de la Narrativa</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                El "Verbo" o "Palabra" (Logos) es el motor de la narrativa bíblica. En el Evangelio de Juan, Jesús es identificado como la Palabra que se hizo carne.

                La Palabra no es simplemente un medio de comunicación, sino una fuerza creadora y redentora.

              </p>

              <p class="text-gray-700 leading-relaxed">

                La Biblia, en su totalidad, es la historia de la Palabra de Dios en el mundo. La Palabra crea el mundo, juzga al mundo y redime al mundo.

              </p>

            </div>

          </div>

        </div>

        <h2 class="font-serif text-4xl font-bold text-gray-800 mb-12">7. Influencia en la Literatura: La Biblia como Fuente de la Imaginación</h2>


        <div id="escritura-secular" class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">7.1 La Biblia como Origen de la Literatura Secular</h3>


          <div class="grid grid-cols-1 lg:grid-cols-3 gap-8 mb-8">

            <div class="lg:col-span-2">

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">7.1.1 La Biblia como "Escritura Secular"</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye argumenta que la Biblia es la fuente de la "escritura secular", es decir, de la literatura que no es explícitamente religiosa.

                La Biblia ha proporcionado los modelos para muchos de los géneros literarios occidentales, como la epopeya, la tragedia, la comedia y el romance.

              </p>

              <p class="text-gray-700 leading-relaxed">

                Los temas y símbolos de la Biblia, como la creación, la caída, el exilio y la redención, se repiten y se reelaboran en la literatura secular.

              </p>

            </div>

            <div class="quote-highlight p-6 rounded-lg">

              <p class="font-serif text-lg italic text-gray-800 mb-2">

                "La Biblia ha proporcionado el 'gran código' que subyace a muchas obras de la literatura occidental."

              </p>

            </div>

          </div>


          <div class="mb-8">

            <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">7.1.2 Influencia en Géneros Literarios como el Romance</h4>

            <p class="text-gray-700 leading-relaxed mb-4">

              Frye dedica una gran parte de su obra al análisis de la influencia de la Biblia en el género del romance.

              El romance, con su énfasis en la aventura, la prueba y la recompensa, tiene sus raíces en las historias de los héroes bíblicos.

            </p>

            <p class="text-gray-700 leading-relaxed">

              El héroe del romance, como el héroe bíblico, es un ser excepcional que se enfrenta a una serie de pruebas y que,

              a través de su valentía y su fe, logra superarlas y alcanzar una recompensa

              <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[522]</a>.

            </p>

          </div>

        </div>


        <div id="teoria-arquetipos">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">7.2 La Biblia y la Crítica Literaria</h3>


          <div class="grid grid-cols-1 lg:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">7.2.1 Aplicación de la Teoría de los Arquetipos de Frye</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                La teoría de los arquetipos de Frye, tal como se presenta en <em>Anatomía de la crítica</em>, es una herramienta fundamental

                para el análisis de la influencia de la Biblia en la literatura. Los arquetipos, según Frye, son símbolos o imágenes que se repiten

                en la literatura y que tienen un significado universal.

              </p>

              <p class="text-gray-700 leading-relaxed">

                La Biblia, como la fuente primaria de la mitología occidental, es un repositorio de arquetipos que han sido utilizados

                por innumerables escritores a lo largo de los siglos.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">7.2.2 La Biblia como Clave para Interpretar la Literatura Occidental</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Para Frye, la Biblia es la clave para interpretar la literatura occidental. La Biblia ha proporcionado el marco de referencia

                narrativo y simbólico que ha permitido a los escritores crear nuevas y originales obras de arte.

              </p>

              <p class="text-gray-700 leading-relaxed">

                La influencia de la Biblia es tan profunda que, para Frye, resulta imposible entender plenamente la literatura occidental

                sin una comprensión de sus estructuras y temas bíblicos subyacentes

                <a href="https://classicalchristian.org/classis/the-bible-northrop-frye-classical-christian-education-part-i/" class="text-primary hover:underline" target="_blank">[334]</a>.

              </p>

            </div>

        function initializeMermaidControls() {

            const containers = document.querySelectorAll('.mermaid-container');


            containers.forEach(container => {

            const mermaidElement = container.querySelector('.mermaid');

            let scale = 1;

            let isDragging = false;

            let startX, startY, translateX = 0, translateY = 0;


            // 触摸相关状态

            let isTouch = false;

            let touchStartTime = 0;

            let initialDistance = 0;

            let initialScale = 1;

            let isPinching = false;


            // Zoom controls

            const zoomInBtn = container.querySelector('.zoom-in');

            const zoomOutBtn = container.querySelector('.zoom-out');

            const resetBtn = container.querySelector('.reset-zoom');

            const fullscreenBtn = container.querySelector('.fullscreen');


            function updateTransform() {

                mermaidElement.style.transform = `translate(${translateX}px, ${translateY}px) scale(${scale})`;


                if (scale > 1) {

                container.classList.add('zoomed');

                } else {

                container.classList.remove('zoomed');

                }


                mermaidElement.style.cursor = isDragging ? 'grabbing' : 'grab';

            }


            if (zoomInBtn) {

                zoomInBtn.addEventListener('click', () => {

                scale = Math.min(scale * 1.25, 4);

                updateTransform();

                });

            }


            if (zoomOutBtn) {

                zoomOutBtn.addEventListener('click', () => {

                scale = Math.max(scale / 1.25, 0.3);

                if (scale <= 1) {

                    translateX = 0;

                    translateY = 0;

                }

                updateTransform();

                });

            }


            if (resetBtn) {

                resetBtn.addEventListener('click', () => {

                scale = 1;

                translateX = 0;

                translateY = 0;

                updateTransform();

                });

            }


            if (fullscreenBtn) {

                fullscreenBtn.addEventListener('click', () => {

                if (container.requestFullscreen) {

                    container.requestFullscreen();

                } else if (container.webkitRequestFullscreen) {

                    container.webkitRequestFullscreen();

                } else if (container.msRequestFullscreen) {

                    container.msRequestFullscreen();

                }

                });

            }


            // Mouse Events

            mermaidElement.addEventListener('mousedown', (e) => {

                if (isTouch) return; // 如果是触摸设备，忽略鼠标事件


                isDragging = true;

                startX = e.clientX - translateX;

                startY = e.clientY - translateY;

                mermaidElement.style.cursor = 'grabbing';

                updateTransform();

                e.preventDefault();

            });


            document.addEventListener('mousemove', (e) => {

                if (isDragging && !isTouch) {

                translateX = e.clientX - startX;

                translateY = e.clientY - startY;

                updateTransform();

                }

            });


            document.addEventListener('mouseup', () => {

                if (isDragging && !isTouch) {

                isDragging = false;

                mermaidElement.style.cursor = 'grab';

                updateTransform();

                }

            });


            document.addEventListener('mouseleave', () => {

                if (isDragging && !isTouch) {

                isDragging = false;

                mermaidElement.style.cursor = 'grab';

                updateTransform();

                }

            });


            // 获取两点之间的距离

            function getTouchDistance(touch1, touch2) {

                return Math.hypot(

                touch2.clientX - touch1.clientX,

                touch2.clientY - touch1.clientY

                );

            }


            // Touch Events - 触摸事件处理

            mermaidElement.addEventListener('touchstart', (e) => {

                isTouch = true;

                touchStartTime = Date.now();


                if (e.touches.length === 1) {

                // 单指拖动

                isPinching = false;

                isDragging = true;


                const touch = e.touches[0];

                startX = touch.clientX - translateX;

                startY = touch.clientY - translateY;


                } else if (e.touches.length === 2) {

                // 双指缩放

                isPinching = true;

                isDragging = false;


                const touch1 = e.touches[0];

                const touch2 = e.touches[1];

                initialDistance = getTouchDistance(touch1, touch2);

                initialScale = scale;

                }


                e.preventDefault();

            }, { passive: false });


            mermaidElement.addEventListener('touchmove', (e) => {

                if (e.touches.length === 1 && isDragging && !isPinching) {

                // 单指拖动

                const touch = e.touches[0];

                translateX = touch.clientX - startX;

                translateY = touch.clientY - startY;

                updateTransform();


                } else if (e.touches.length === 2 && isPinching) {

                // 双指缩放

                const touch1 = e.touches[0];

                const touch2 = e.touches[1];

                const currentDistance = getTouchDistance(touch1, touch2);


                if (initialDistance > 0) {

                    const newScale = Math.min(Math.max(

                    initialScale * (currentDistance / initialDistance),

                    0.3

                    ), 4);

                    scale = newScale;

                    updateTransform();

                }

                }


                e.preventDefault();

            }, { passive: false });


            mermaidElement.addEventListener('touchend', (e) => {

                // 重置状态

                if (e.touches.length === 0) {

                isDragging = false;

                isPinching = false;

                initialDistance = 0;


                // 延迟重置isTouch，避免鼠标事件立即触发

                setTimeout(() => {

                    isTouch = false;

                }, 100);

                } else if (e.touches.length === 1 && isPinching) {

                // 从双指变为单指，切换为拖动模式

                isPinching = false;

                isDragging = true;


                const touch = e.touches[0];

                startX = touch.clientX - translateX;

                startY = touch.clientY - translateY;

                }


                updateTransform();

            });


            mermaidElement.addEventListener('touchcancel', (e) => {

                isDragging = false;

                isPinching = false;

                initialDistance = 0;


                setTimeout(() => {

                isTouch = false;

                }, 100);


                updateTransform();

            });


            // Enhanced wheel zoom with better center point handling

            container.addEventListener('wheel', (e) => {

                e.preventDefault();

                const rect = container.getBoundingClientRect();

                const centerX = rect.width / 2;

                const centerY = rect.height / 2;


                const delta = e.deltaY > 0 ? 0.9 : 1.1;

                const newScale = Math.min(Math.max(scale * delta, 0.3), 4);


                // Adjust translation to zoom towards center

                if (newScale !== scale) {

                const scaleDiff = newScale / scale;

                translateX = translateX * scaleDiff;

                translateY = translateY * scaleDiff;

                scale = newScale;


                if (scale <= 1) {

                    translateX = 0;

                    translateY = 0;

                }


                updateTransform();

                }

            });


            // Initialize display

            updateTransform();

            });

        }


        // Initialize mermaid controls when DOM is loaded

        document.addEventListener('DOMContentLoaded', function() {

            initializeMermaidControls();

        });

                    });

                }

            });

        });

              los eventos históricos.

            <p class="text-gray-700 leading-relaxed mb-4">

              En lugar de abordar la Biblia desde una perspectiva teológica o histórica, Frye la examina como un crítico literario,

              interesado en su unidad interna y su poder como un sistema de mitos y metáforas

              <a href="https://www.dialoguejournal.com/articles/the-great-code-revealed-bible-and-literature-by-northrop-frye/" class="text-primary hover:underline" target="_blank">[519]</a>.

            </p>

          </div>

        <div id="estructura-quiasica" class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">1.2 Unidad Estructural y Narrativa</h3>


          <div class="mb-8">

            <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">1.2.1 Estructura Quíasica o de &#34;Doble Espejo&#34;</h4>

            <p class="text-gray-700 leading-relaxed mb-4">

              El libro de Frye se divide en dos partes: <strong>&#34;El orden de las palabras&#34; y &#34;El orden de los tipos&#34;</strong>

              <a href="https://www.dialoguejournal.com/articles/the-great-code-revealed-bible-and-literature-by-northrop-frye/" class="text-primary hover:underline" target="_blank">[519]</a>.

              La primera parte explora los principios básicos de interpretación a través de los conceptos de lenguaje, mito, metáfora y tipología.

              La segunda parte revisita estos mismos cuatro temas, pero en orden inverso.

            </p>

            <div class="quote-highlight p-6 rounded-lg mb-6">

              <p class="font-serif text-lg italic text-gray-800 mb-2">

                &#34;Esta disposición crea una estructura quíasica, o de &#39;doble espejo&#39;, donde cada mitad del libro refleja y revela el significado de la otra.&#34;

              </p>

              <cite class="text-sm text-gray-600">

                — <a href="https://macblog.mcmaster.ca/fryeblog/what-the-great-code-is-and-does/" class="text-primary hover:underline" target="_blank">[524]</a>

              </cite>

            </div>

          </div>


          <div class="mb-8">

            <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">1.2.2 Coherencia entre el Antiguo y el Nuevo Testamento</h4>

            <p class="text-gray-700 leading-relaxed mb-4">

              La unidad de la Biblia se manifiesta en la coherencia entre los dos testamentos. A través de su método tipológico,

              Frye demuestra que los dos testamentos no son colecciones independientes de textos, sino partes de una sola narrativa

              que se refleja y se completa a sí misma

              <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[502]</a>.

            </p>

            <div class="quote-highlight p-6 rounded-lg">

              <p class="font-serif text-lg italic text-gray-800 mb-2">

                &#34;El Antiguo Testamento se revela en el Nuevo, y el Nuevo Testamento está oculto en el Antiguo,

                creando una &#39;doble mirada&#39; en la que cada parte refleja la otra.&#34;

              </p>

              <cite class="text-sm text-gray-600">

                — <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[37]</a>

              </cite>

            </div>

        <div id="mitologia-occidental">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">1.3 La Biblia como Fuente de Mitología Occidental</h3>


          <div class="grid grid-cols-1 md:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">1.3.1 Influencia en la Literatura y el Arte Seculares</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye sostiene que la Biblia ha sido la fuente primaria de la mitología occidental, proporcionando los arquetipos

                y los símbolos que han alimentado la imaginación de los artistas y escritores a lo largo de los siglos.

                En <em>The Secular Scripture</em>, Frye explora cómo la literatura secular ha recreado y secularizado los mitos bíblicos

                <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[502]</a>.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">1.3.2 La Biblia como &#34;Mito del Hombre Occidental&#34;</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Para Frye, la Biblia es el <strong>&#34;mito del hombre occidental&#34;</strong>, una narrativa que define la identidad

                y el destino de la humanidad desde una perspectiva occidental. A diferencia de otros mitos que se centran en dioses

                y héroes, el mito bíblico es una historia sobre la humanidad

                <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[516]</a>.

              </p>

            </div>

          </div>

        </div>

      </section>

      <!-- Section 2: Kerygma -->

      <section id="kerygma" class="max-w-7xl mx-auto px-6 py-16">

        <h2 class="font-serif text-4xl font-bold text-gray-800 mb-12">2. Kerygma: El Lenguaje Único de la Biblia</h2>


        <div id="kerygma-definicion" class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">2.1 Definición de Kerygma como &#34;Proclamación&#34;</h3>


          <div class="grid grid-cols-1 lg:grid-cols-3 gap-8">

            <div class="lg:col-span-2">

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye adopta el término griego <em>kerygma</em>, que significa &#34;proclamación&#34; o &#34;predicación&#34;, y le otorga un significado

                mucho más profundo y específico en el contexto del lenguaje bíblico

                <a href="https://macblog.mcmaster.ca/fryeblog/what-the-great-code-is-and-does/" class="text-primary hover:underline" target="_blank">[384]</a>.

                Para él, <em>kerygma</em> no es simplemente el contenido del mensaje cristiano, sino la forma lingüística única

                a través de la cual se transmite la revelación divina.

              </p>

              <p class="text-gray-700 leading-relaxed mb-6">

                Este lenguaje no es argumentativo ni persuasivo en el sentido retórico convencional; más bien, es un acto de habla

                que crea una nueva realidad para el oyente. Dios y Jesús, en sus expresiones de preocupación, utilizan metáforas,

                pero no se comprometen en argumentaciones.

              </p>

            </div>

            <div class="quote-highlight p-6 rounded-lg">

              <p class="font-serif text-lg italic text-gray-800 mb-2">

                &#34;El kerygma es un acto místico y mistagógico que tiene el poder de iniciar al oyente en una realidad espiritual superior.&#34;

              </p>

              <cite class="text-sm text-gray-600">

                — <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[437]</a>

              </cite>

            </div>

          </div>


          <div class="grid grid-cols-1 md:grid-cols-2 gap-8 mt-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">2.1.1 Kerygma como Acto Místico y Mistagógico</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Para Frye, el <em>kerygma</em> es mucho más que una simple proclamación verbal; es un <strong>acto místico y mistagógico</strong>

                que tiene el poder de iniciar al oyente en una realidad espiritual superior. El término &#34;mistagógico&#34; se refiere a la

                iniciación en los misterios.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">2.1.2 Revelación de una Realidad Espiritual</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                El <em>kerygma</em> es el principal mecanismo a través del cual la Biblia revela una realidad espiritual que trasciende

                el mundo físico. El lenguaje <em>kerygmatico</em> no describe esta realidad objetivamente; más bien, la &#34;revela&#34; al traerla

                a la existencia en la imaginación y la conciencia del oyente

                <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[75]</a>.

              </p>

            </div>

          </div>

        </div>


        <div class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">2.2 Kerygma vs. Lenguaje Literal y Descriptivo</h3>


          <div class="mb-8">

            <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">2.2.1 Verdad Poética vs. Verdad Literal</h4>

            <p class="text-gray-700 leading-relaxed mb-4">

              Frye argumenta que la Biblia no debe ser leída como un documento histórico o científico que pretende ofrecer una

              descripción literal y objetiva de los hechos. Intentar leer la Biblia de esta manera es caer en el &#34;literalismo&#34;,

              una forma de anti-intelectualismo que malinterpreta la naturaleza del lenguaje bíblico.

            </p>

          </div>


          <div class="mb-8">

            <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">2.2.2 Transcendencia de la División Sujeto/Objeto</h4>

            <p class="text-gray-700 leading-relaxed mb-4">

              Una de las características más importantes del <em>kerygma</em> es su capacidad para <strong>trascender la división entre sujeto y objeto</strong>

              que caracteriza al lenguaje descriptivo. El <em>kerygma</em> es un lenguaje que &#34;nos posee&#34;, más que nosotros poseerlo,

              y en el que la separación entre el habla activa y la recepción de la palabra desaparece

              <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[516]</a>.

            </p>

          </div>

        </div>


        <div id="kerygma-fases">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">2.3 Kerygma en el Marco de las Fases del Lenguaje de Vico</h3>


          <div class="grid grid-cols-1 lg:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">2.3.1 Posicionamiento de Kerygma como Fase Superior</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Para situar el <em>kerygma</em> en el contexto de la historia del lenguaje, Frye se sirve de la teoría de las tres fases

                del lenguaje de Giambattista Vico: la fase poética, la fase heroica y la fase vulgar

                <a href="https://macblog.mcmaster.ca/fryeblog/2009/09/24/the-phases-and-modes-of-language/" class="text-primary hover:underline" target="_blank">[525]</a>.

              </p>

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye propone el <em>kerygma</em> como una cuarta forma de expresión, o incluso como una quinta fase que se sitúa

                <strong>&#34;al otro lado de lo poético&#34;</strong>

                <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[535]</a>.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">2.3.2 Contraste con la Desmitologización de Bultmann</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                La postura de Frye se ve reforzada por su rechazo a la teoría de la <strong>&#34;demitologización&#34;</strong> del teólogo Rudolf Bultmann.

                Mientras Bultmann argumentaba que el mensaje del Nuevo Testamento debía ser separado de su envoltura mítica,

                Frye insiste en que el mito es el lenguaje apropiado y necesario para expresar las verdades del <em>kerygma</em>

                <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[530]</a>.

              </p>

            </div>

          </div>

        </div>

      </section>

        <h2 class="font-serif text-4xl font-bold text-gray-800 mb-12">3. Orden de Tipos: La Tipología como Método Interpretativo</h2>


        <div id="estructura-unificadora" class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">3.1 Tipología como Estructura Unificadora de la Biblia</h3>


          <div class="grid grid-cols-1 lg:grid-cols-2 gap-8 mb-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">3.1.1 Relación entre Tipo y Antitipo</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                La tipología es el método interpretativo que Frye utiliza para demostrar la unidad estructural de la Biblia.

                <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[502]</a>.

              </p>

              <p class="text-gray-700 leading-relaxed">

                La relación entre el tipo y el antitipo no es simplemente una cuestión de predicción y cumplimiento,

                sino de una <strong>&#34;doble mirada&#34; en la que cada testamento refleja y da sentido al otro</strong>

                <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[37]</a>.

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">3.2 Tipología y la Tensión entre Literatura y Teología</h3>


          <div class="grid grid-cols-1 md:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">3.2.1 Uso de la Tipología como Herramienta Literaria</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Aunque Frye insiste en que su enfoque de la Biblia es literario y no teológico, su uso de la tipología lo sitúa

                en una zona de tensión entre ambos campos. Frye adopta la tipología como una herramienta para analizar la estructura

                y el significado de la Biblia como un texto literario.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">3.2.2 Reconocimiento de las Raíces Teológicas</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                A pesar de su enfoque literario, Frye reconoce que la tipología no puede disociarse completamente de su fundamento teológico.

                Los críticos señalan que su hermenéutica bíblica comparte principios fundamentales con muchos teólogos cristianos modernos

                <a href="https://dialnet.unirioja.es/descarga/articulo/1271431.pdf" class="text-primary hover:underline" target="_blank">[1]</a>.

      <section id="mitopoesia" class="max-w-7xl mx-auto px-6 py-16">

        <h2 class="font-serif text-4xl font-bold text-gray-800 mb-12">4. Mitopoesía vs. Historia: La Biblia como Mito, no como Crónica</h2>


        <div id="narrativa-mitica" class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">4.1 La Biblia como Narrativa Mítica</h3>


          <div class="grid grid-cols-1 lg:grid-cols-2 gap-8 mb-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">4.1.1 Énfasis en el Significado sobre la Facticidad</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Para Frye, la Biblia debe ser entendida primordialmente como una narrativa mítica, no como un documento histórico.

                El término &#34;mito&#34;, en su uso, no implica falsedad, sino una secuencia de palabras cargada de una seriedad y una importancia especiales

                <a href="https://www.dialoguejournal.com/articles/the-great-code-revealed-bible-and-literature-by-northrop-frye/" class="text-primary hover:underline" target="_blank">[69]</a>.

              </p>

              <p class="text-gray-700 leading-relaxed">

                El valor de la Biblia no radica en su veracidad histórica, sino en su capacidad para transmitir verdades profundas

                sobre la condición humana y su relación con lo divino.

              </p>

            </div>

            <div class="quote-highlight p-6 rounded-lg">

              <p class="font-serif text-lg italic text-gray-800 mb-2">

                &#34;Comparar los mitos de diluvio con inundaciones históricas es menos fructífero que compararlos con otros mitos de diluvio,

                ya que su significado reside en su estructura simbólica, no en su facticidad.&#34;

              </p>

              <cite class="text-sm text-gray-600">

                — <a href="https://tntlab.carterschool.gmu.edu/learn/narrative-transformation-resources/narrative-themography/myth/" class="text-primary hover:underline" target="_blank">[78]</a>

              </cite>

            </div>

          </div>


          <div class="mb-8">

            <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">4.1.2 &#34;Heilsgeschichte&#34; vs. &#34;Weltgeschichte&#34;</h4>

            <p class="text-gray-700 leading-relaxed mb-4">

              Frye distingue entre dos tipos de historia: la <em>Heilsgeschichte</em> (historia de la salvación) y la

              <em>Weltgeschichte</em> (historia del mundo). La Biblia pertenece a la primera categoría.

            </p>

            <p class="text-gray-700 leading-relaxed">

              La <em>Weltgeschichte</em> es la historia de los eventos tal como ocurrieron, una crónica de hechos sin un propósito trascendental.

              La <em>Heilsgeschichte</em>, por otro lado, es una narrativa de redención, una historia que tiene un sentido y una dirección

              <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[37]</a>.

            </p>

          </div>

        </div>


        <div id="critica-historico">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">4.2 Crítica al Enfoque Histórico-Crítico</h3>


          <div class="grid grid-cols-1 md:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">4.2.1 Rechazo de la Lectura como Documento Histórico</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye rechaza el enfoque histórico-crítico de la Biblia, que busca establecer la veracidad histórica de sus relatos

                a través del análisis de fuentes y la comparación con otros documentos históricos. Para Frye, este enfoque es inadecuado

                porque ignora la naturaleza mítica y simbólica de la Biblia.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">4.2.2 Defensa de un Enfoque Literario y Simbólico</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                En lugar del enfoque histórico-crítico, Frye defiende un enfoque literario y simbólico de la Biblia.

                Este enfoque se centra en el análisis de la estructura narrativa, los patrones de imágenes y los símbolos de la Biblia,

                con el objetivo de revelar su unidad y coherencia como una obra de arte.

              </p>

            </div>

          </div>

        </div>

      </section>


      <div class="section-divider max-w-7xl mx-auto"></div>

      <section id="arquetipos" class="max-w-7xl mx-auto px-6 py-16">

        <h2 class="font-serif text-4xl font-bold text-gray-800 mb-12">5. Imagen y Arquetipo: Los Símbolos Visuales de la Biblia</h2>


        <div id="sistemas-imagineria" class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">5.1 Sistemas de Imaginería Unificados</h3>


          <div class="grid grid-cols-1 lg:grid-cols-2 gap-8 mb-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">5.1.1 Imaginería Apocalíptica vs. Imaginería Demoníaca</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye identifica dos sistemas de imaginería opuestos y unificados en la Biblia: la <strong>imaginería apocalíptica</strong>

                y la <strong>imaginería demoníaca</strong>. La imaginería apocalíptica representa el mundo de la creación, el paraíso y la redención,

                mientras que la imaginería demoníaca representa el mundo de la caída, el caos y la destrucción.

              </p>

            </div>

            <div>

              <img src="https://kimi-web-img.moonshot.cn/img/static.wikia.nocookie.net/6e1386437b2dfce9b8bb8b074d265a574218b70d" alt="Biblical imagery contrast: heavenly paradise versus chaotic destruction" class="w-full h-64 object-cover rounded-lg shadow-lg" size="medium" aspect="wide" query="biblical paradise and chaos contrast" referrerpolicy="no-referrer" data-modified="1" data-score="0.00"/>

            </div>

          </div>


          <div class="quote-highlight p-6 rounded-lg mb-8">

            <p class="font-serif text-lg italic text-gray-800 mb-2">

              &#34;Estos dos sistemas de imaginería no son simplemente opuestos, sino que están intrínsecamente relacionados.

              La imaginería demoníaca es una perversión de la imaginería apocalíptica, y la imaginería apocalíptica

              es la restauración de la imaginería demoníaca.&#34;

            </p>

          </div>

        </div>


        <div id="naturaleza-simbolo">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">5.2 La Naturaleza como Símbolo</h3>


          <div class="grid grid-cols-1 md:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">5.2.1 La Creación como Orden Divino</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                En la Biblia, la naturaleza no es simplemente un fondo para la acción humana, sino un símbolo del orden divino.

                La creación, tal como se describe en el Génesis, es un acto de dar forma y orden al caos.

              </p>

              <p class="text-gray-700 leading-relaxed">

                El jardín del Edén es el símbolo supremo de esta armonía entre la naturaleza y lo divino

                <a href="https://classicalchristian.org/classis/the-bible-northrop-frye-classical-christian-education-part-i/" class="text-primary hover:underline" target="_blank">[187]</a>.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">5.2.2 La Caída como Caos y Desorden</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                La caída del hombre no es simplemente un evento histórico, sino un símbolo de la entrada del caos y el desorden en el mundo.

                La naturaleza, que antes era un reflejo del orden divino, se convierte en un lugar de alienación y sufrimiento.

              </p>

              <p class="text-gray-700 leading-relaxed">

                El desierto, con su sequedad y su soledad, se convierte en el símbolo supremo de la naturaleza caída.

              </p>

            </div>

          </div>

        </div>

      </section>

      <!-- Section 6: Lenguaje y Prosa -->

      <section id="lenguaje-prosaico" class="max-w-7xl mx-auto px-6 py-16">

        <h2 class="font-serif text-4xl font-bold text-gray-800 mb-12">6. La Palabra y la Prosa: La Estructura Lingüística de la Biblia</h2>


        <div class="grid grid-cols-1 lg:grid-cols-2 gap-12 mb-16">

          <div>

            <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">6.1 El Lenguaje de la Biblia como &#34;Prosa&#34;</h3>


            <div class="mb-8">

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">6.1.1 Actos de Habla Prosaicos</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye argumenta que el lenguaje de la Biblia es esencialmente prosaico, en el sentido de que está compuesto por actos de habla

                que tienen un propósito específico. Los actos de habla prosaicos en la Biblia incluyen mandamientos, exhortaciones, oráculos y proclamaciones.

              </p>

              <p class="text-gray-700 leading-relaxed">

                La Biblia es una obra de &#34;literatura plus&#34;, un texto que posee una gravedad y una seriedad que lo distinguen de la ficción puramente hipotética

                <a href="https://spectrum.library.concordia.ca/994328/6/Roseman_MA_F2024.pdf" class="text-primary hover:underline" target="_blank">[179]</a>.

              </p>

            </div>

          </div>

          <div>

            <img src="https://kimi-web-img.moonshot.cn/img/brewminate.com/f32d0ed01cc767859c1634dd3149eff48dbaea4b.png" alt="Ancient biblical manuscript with calligraphy" class="w-full h-80 object-cover rounded-lg shadow-lg" size="medium" aspect="wide" style="photo" query="ancient biblical manuscript calligraphy" referrerpolicy="no-referrer" data-modified="1" data-score="0.00"/>

          </div>

        </div>


        <div class="mb-8">

          <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">6.1.2 Distinción entre Sentido Literal y Sentido Verbal</h4>

          <p class="text-gray-700 leading-relaxed mb-4">

            Frye hace una distinción fundamental entre el sentido literal y el sentido verbal de la Biblia.

            El sentido literal se refiere a la correspondencia entre las palabras y los hechos históricos.

            El sentido verbal se refiere al significado que surge de las conexiones y patrones internos del texto.

          </p>

          <p class="text-gray-700 leading-relaxed">

            Frye argumenta que el verdadero sentido &#34;literal&#34; de la Biblia es su sentido verbal, no su referencia a eventos históricos externos.

          </p>

        </div>


        <div class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">6.2 La Narrativa de la Biblia: De la Creación al Apocalipsis</h3>


          <div class="grid grid-cols-1 md:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">6.2.1 Estructura Narrativa Circular</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                La narrativa de la Biblia, según Frye, tiene una estructura circular que va desde la creación hasta el apocalipsis.

                El libro comienza con la creación del mundo en el Génesis y termina con la creación de un nuevo cielo y una nueva tierra en el Apocalipsis.

              </p>

              <p class="text-gray-700 leading-relaxed">

                Esta estructura circular sugiere que la historia no es una línea recta, sino un ciclo de caída y redención.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">6.2.2 El &#34;Verbo&#34; como Motor de la Narrativa</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                El &#34;Verbo&#34; o &#34;Palabra&#34; (Logos) es el motor de la narrativa bíblica. En el Evangelio de Juan, Jesús es identificado como la Palabra que se hizo carne.

                La Palabra no es simplemente un medio de comunicación, sino una fuerza creadora y redentora.

              </p>

              <p class="text-gray-700 leading-relaxed">

                La Biblia, en su totalidad, es la historia de la Palabra de Dios en el mundo. La Palabra crea el mundo, juzga al mundo y redime al mundo.

              </p>

            </div>

          </div>

        </div>

      </section>

      <section id="influencia-literaria" class="max-w-7xl mx-auto px-6 py-16">

        <h2 class="font-serif text-4xl font-bold text-gray-800 mb-12">7. Influencia en la Literatura: La Biblia como Fuente de la Imaginación</h2>


        <div id="escritura-secular" class="mb-16">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">7.1 La Biblia como Origen de la Literatura Secular</h3>


          <div class="grid grid-cols-1 lg:grid-cols-3 gap-8 mb-8">

            <div class="lg:col-span-2">

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">7.1.1 La Biblia como &#34;Escritura Secular&#34;</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Frye argumenta que la Biblia es la fuente de la &#34;escritura secular&#34;, es decir, de la literatura que no es explícitamente religiosa.

                La Biblia ha proporcionado los modelos para muchos de los géneros literarios occidentales, como la epopeya, la tragedia, la comedia y el romance.

              </p>

              <p class="text-gray-700 leading-relaxed">

                Los temas y símbolos de la Biblia, como la creación, la caída, el exilio y la redención, se repiten y se reelaboran en la literatura secular.

              </p>

            </div>

            <div class="quote-highlight p-6 rounded-lg">

              <p class="font-serif text-lg italic text-gray-800 mb-2">

                &#34;La Biblia ha proporcionado el &#39;gran código&#39; que subyace a muchas obras de la literatura occidental.&#34;

              </p>

            </div>

          </div>


          <div class="mb-8">

            <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">7.1.2 Influencia en Géneros Literarios como el Romance</h4>

            <p class="text-gray-700 leading-relaxed mb-4">

              Frye dedica una gran parte de su obra al análisis de la influencia de la Biblia en el género del romance.

              El romance, con su énfasis en la aventura, la prueba y la recompensa, tiene sus raíces en las historias de los héroes bíblicos.

            </p>

            <p class="text-gray-700 leading-relaxed">

              El héroe del romance, como el héroe bíblico, es un ser excepcional que se enfrenta a una serie de pruebas y que,

              a través de su valentía y su fe, logra superarlas y alcanzar una recompensa

              <a href="https://www.um.es/tonosdigital/znum23/secciones/estudios-3-frey.htm" class="text-primary hover:underline" target="_blank">[522]</a>.

            </p>

          </div>

        </div>


        <div id="teoria-arquetipos">

          <h3 class="font-serif text-2xl font-semibold text-gray-800 mb-6">7.2 La Biblia y la Crítica Literaria</h3>


          <div class="grid grid-cols-1 lg:grid-cols-2 gap-8">

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">7.2.1 Aplicación de la Teoría de los Arquetipos de Frye</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                La teoría de los arquetipos de Frye, tal como se presenta en <em>Anatomía de la crítica</em>, es una herramienta fundamental

                para el análisis de la influencia de la Biblia en la literatura. Los arquetipos, según Frye, son símbolos o imágenes que se repiten

                en la literatura y que tienen un significado universal.

              </p>

              <p class="text-gray-700 leading-relaxed">

                La Biblia, como la fuente primaria de la mitología occidental, es un repositorio de arquetipos que han sido utilizados

                por innumerables escritores a lo largo de los siglos.

              </p>

            </div>

            <div>

              <h4 class="font-serif text-xl font-semibold text-gray-800 mb-4">7.2.2 La Biblia como Clave para Interpretar la Literatura Occidental</h4>

              <p class="text-gray-700 leading-relaxed mb-4">

                Para Frye, la Biblia es la clave para interpretar la literatura occidental. La Biblia ha proporcionado el marco de referencia

                narrativo y simbólico que ha permitido a los escritores crear nuevas y originales obras de arte.

              </p>

              <p class="text-gray-700 leading-relaxed">

                La influencia de la Biblia es tan profunda que, para Frye, resulta imposible entender plenamente la literatura occidental

                sin una comprensión de sus estructuras y temas bíblicos subyacentes

                <a href="https://classicalchristian.org/classis/the-bible-northrop-frye-classical-christian-education-part-i/" class="text-primary hover:underline" target="_blank">[334]</a>.

              </p>

            </div>

        // Initialize Mermaid Controls for zoom and pan

        function initializeMermaidControls() {

            const containers = document.querySelectorAll('.mermaid-container');


            containers.forEach(container => {

            const mermaidElement = container.querySelector('.mermaid');

            let scale = 1;

            let isDragging = false;

            let startX, startY, translateX = 0, translateY = 0;


            // 触摸相关状态

            let isTouch = false;

            let touchStartTime = 0;

            let initialDistance = 0;

            let initialScale = 1;

            let isPinching = false;


            // Zoom controls

            const zoomInBtn = container.querySelector('.zoom-in');

            const zoomOutBtn = container.querySelector('.zoom-out');

            const resetBtn = container.querySelector('.reset-zoom');

            const fullscreenBtn = container.querySelector('.fullscreen');


            function updateTransform() {

                mermaidElement.style.transform = `translate(${translateX}px, ${translateY}px) scale(${scale})`;


                if (scale > 1) {

                container.classList.add('zoomed');

                } else {

                container.classList.remove('zoomed');

                }


                mermaidElement.style.cursor = isDragging ? 'grabbing' : 'grab';

            }


            if (zoomInBtn) {

                zoomInBtn.addEventListener('click', () => {

                scale = Math.min(scale * 1.25, 4);

                updateTransform();

                });

            }


            if (zoomOutBtn) {

                zoomOutBtn.addEventListener('click', () => {

                scale = Math.max(scale / 1.25, 0.3);

                if (scale <= 1) {

                    translateX = 0;

                    translateY = 0;

                }

                updateTransform();

                });

            }


            if (resetBtn) {

                resetBtn.addEventListener('click', () => {

                scale = 1;

                translateX = 0;

                translateY = 0;

                updateTransform();

                });

            }


            if (fullscreenBtn) {

                fullscreenBtn.addEventListener('click', () => {

                if (container.requestFullscreen) {

                    container.requestFullscreen();

                } else if (container.webkitRequestFullscreen) {

                    container.webkitRequestFullscreen();

                } else if (container.msRequestFullscreen) {

                    container.msRequestFullscreen();

                }

                });

            }


            // Mouse Events

            mermaidElement.addEventListener('mousedown', (e) => {

                if (isTouch) return; // 如果是触摸设备，忽略鼠标事件


                isDragging = true;

                startX = e.clientX - translateX;

                startY = e.clientY - translateY;

                mermaidElement.style.cursor = 'grabbing';

                updateTransform();

                e.preventDefault();

            });


            document.addEventListener('mousemove', (e) => {

                if (isDragging && !isTouch) {

                translateX = e.clientX - startX;

                translateY = e.clientY - startY;

                updateTransform();

                }

            });


            document.addEventListener('mouseup', () => {

                if (isDragging && !isTouch) {

                isDragging = false;

                mermaidElement.style.cursor = 'grab';

                updateTransform();

                }

            });


            document.addEventListener('mouseleave', () => {

                if (isDragging && !isTouch) {

                isDragging = false;

                mermaidElement.style.cursor = 'grab';

                updateTransform();

                }

            });


            // 获取两点之间的距离

            function getTouchDistance(touch1, touch2) {

                return Math.hypot(

                touch2.clientX - touch1.clientX,

                touch2.clientY - touch1.clientY

                );

            }


            // Touch Events - 触摸事件处理

            mermaidElement.addEventListener('touchstart', (e) => {

                isTouch = true;

                touchStartTime = Date.now();


                if (e.touches.length === 1) {

                // 单指拖动

                isPinching = false;

                isDragging = true;


                const touch = e.touches[0];

                startX = touch.clientX - translateX;

                startY = touch.clientY - translateY;


                } else if (e.touches.length === 2) {

                // 双指缩放

                isPinching = true;

                isDragging = false;


                const touch1 = e.touches[0];

                const touch2 = e.touches[1];

                initialDistance = getTouchDistance(touch1, touch2);

                initialScale = scale;

                }


                e.preventDefault();

            }, { passive: false });


            mermaidElement.addEventListener('touchmove', (e) => {

                if (e.touches.length === 1 && isDragging && !isPinching) {

                // 单指拖动

                const touch = e.touches[0];

                translateX = touch.clientX - startX;

                translateY = touch.clientY - startY;

                updateTransform();


                } else if (e.touches.length === 2 && isPinching) {

                // 双指缩放

                const touch1 = e.touches[0];

                const touch2 = e.touches[1];

                const currentDistance = getTouchDistance(touch1, touch2);


                if (initialDistance > 0) {

                    const newScale = Math.min(Math.max(

                    initialScale * (currentDistance / initialDistance),

                    0.3

                    ), 4);

                    scale = newScale;

                    updateTransform();

                }

                }


                e.preventDefault();

            }, { passive: false });


            mermaidElement.addEventListener('touchend', (e) => {

                // 重置状态

                if (e.touches.length === 0) {

                isDragging = false;

                isPinching = false;

                initialDistance = 0;


                // 延迟重置isTouch，避免鼠标事件立即触发

                setTimeout(() => {

                    isTouch = false;

                }, 100);

                } else if (e.touches.length === 1 && isPinching) {

                // 从双指变为单指，切换为拖动模式

                isPinching = false;

                isDragging = true;


                const touch = e.touches[0];

                startX = touch.clientX - translateX;

                startY = touch.clientY - translateY;

                }


                updateTransform();

            });


            mermaidElement.addEventListener('touchcancel', (e) => {

                isDragging = false;

                isPinching = false;

                initialDistance = 0;


                setTimeout(() => {

                isTouch = false;

                }, 100);


                updateTransform();

            });


            // Enhanced wheel zoom with better center point handling

            container.addEventListener('wheel', (e) => {

                e.preventDefault();

                const rect = container.getBoundingClientRect();

                const centerX = rect.width / 2;

                const centerY = rect.height / 2;


                const delta = e.deltaY > 0 ? 0.9 : 1.1;

                const newScale = Math.min(Math.max(scale * delta, 0.3), 4);


                // Adjust translation to zoom towards center

                if (newScale !== scale) {

                const scaleDiff = newScale / scale;

                translateX = translateX * scaleDiff;

                translateY = translateY * scaleDiff;

                scale = newScale;


                if (scale <= 1) {

                    translateX = 0;

                    translateY = 0;

                }


                updateTransform();

                }

            });


            // Initialize display

            updateTransform();

            });

        }


        // Initialize mermaid controls when DOM is loaded

        document.addEventListener('DOMContentLoaded', function() {

            initializeMermaidControls();

        });


        // Smooth scrolling for anchor links

        document.querySelectorAll('a[href^="#"]').forEach(anchor => {

            anchor.addEventListener('click', function (e) {

                e.preventDefault();

                const target = document.querySelector(this.getAttribute('href'));

                if (target) {

                    target.scrollIntoView({

                        behavior: 'smooth',

                        block: 'start'

                    });

                }

            });

        });

    </script>
